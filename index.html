Nombre del producto

Almacenes Muebles del Sur

Sistema contable para microempresa que vende muebles para el hogar,enceres para oficina, registro de ventas, pago a proveedores online, pago de productos, e-comerce.

Características principales
  1. Registro nuevos clientes
  2. creación contraseña
  3. Recuperación contraseña
  4. Venta de muebles y enceres on-line
  5. Diferentes formas de pago
  6. Inclusión sistema contable electrónico
  7. Inclusión de productos a la venta
Beneficios para el usuario

Facilita las formas de pago en línea por la web

Brindar excelente calidad de los productos y también la posibilidad de poder solicitar muebles personalizados

Público objetivo

Facilidad de adquisición de muebles y enceres para el hogar y la oficina

Funciones del producto

  • Permitir registro en la página
  • Restablecimiento de contraseña
  • Ventas en línea e-comerce
  • Control contable
  • Compra y pago a proveedores
